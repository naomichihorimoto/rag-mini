# API仕様書

## 概要
本APIは、文書管理システムへのアクセスを提供します。

## エンドポイント

### GET /api/documents
文書一覧を取得します。

**パラメータ:**
- `page`: ページ番号（オプション、デフォルト: 1）
- `limit`: 1ページあたりの件数（オプション、デフォルト: 20）

**レスポンス例:**
```json
{
  "documents": [
    {
      "id": 1,
      "title": "サンプル文書",
      "content": "文書の内容...",
      "created_at": "2024-01-01T00:00:00Z"
    }
  ],
  "total": 100,
  "page": 1
}
```

### POST /api/documents
新しい文書を作成します。

**リクエスト例:**
```json
{
  "title": "新しい文書",
  "content": "文書の内容..."
}
```

### GET /api/search
文書を検索します。

**パラメータ:**
- `q`: 検索クエリ（必須）
- `limit`: 結果の件数（オプション、デフォルト: 10）
